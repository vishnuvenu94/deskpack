# Roadmap

Near-term priorities for `deskpack`:

1. Strengthen the supported topology matrix with more real-world fixtures and smoke tests.
2. Improve first-class handling for local database workflows, especially SQLite-oriented apps.
3. Investigate a clear support story for Next.js standalone SSR packaging.
4. Investigate TanStack Start runtime server support beyond static and prerendered output.
5. Expand release documentation for signing, notarization boundaries, and platform-specific packaging expectations.
